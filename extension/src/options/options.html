The brand accent used across the popup, side panel, quick crop and these options. Applies instantly and is remembered on this device.
The Stencil browser app served over HTTP (e.g. npm run serve → http://localhost:8080/). The full editor opens in a new tab; the quick crop opens as an in-page modal.
Uses a local record of images this extension has opened. Turn off marking to always open a fresh editor with no badges. These toggles are also available directly in the popup.
Pick the outline colour.
The outline colour used when you tick “highlight on page” in the popup. “Main theme” keeps it in sync with the accent above; the mark/hover ring is derived from it.
Off by default. When on, a scripting API is injected into every page's main world so you can run e.g. stencil.images, stencil.search('logo'), or stencil.open(el) from the DevTools console. Leave off unless you need it.

Server connections

Connect to a Stencil collaboration server. Its stored projects appear in the popup as shared pins (golden outline), and pinning an image lets you store it there too.

A token is validated if you supply one, otherwise one is requested from the server (open issuance / dev mode). Stored in chrome.storage.local.
