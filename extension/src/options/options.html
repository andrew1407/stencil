Stencil Image Picker

Where the editor is served, how it opens, and the default page size.

The Stencil browser app served over HTTP (e.g. npm run serve → http://localhost:8080/). The full editor opens in a new tab; the quick crop opens as an in-page modal.
Uses a local record of images this extension has opened. Turn off marking to always open a fresh editor with no badges. These toggles are also available directly in the popup.
Off by default. When on, a scripting API is injected into every page's main world so you can run e.g. stencil.images, stencil.search('logo'), or stencil.open(el) from the DevTools console. Leave off unless you need it.
